# services-json

> My services list

## Usage
See: [example](example/app.jsx)

```js
const SERVICES_JSON_URL = "https://vovanr.github.io/services-json/dist/services.json";

function fetchServicesJSON() {
  return fetch(SERVICES_JSON_URL).then(resp => resp.json());
}

fetchServicesJSON()
  .then(console.log)
  //=> [{id: '', name: '', desc: '', ...}, ...]
```
